# Keyword Ownership Engine - Frontend Dashboard

**Legendary React + TypeScript + Tailwind CSS Dashboard** 🎯

Production-ready frontend for the Keyword Ownership Engine with beautiful UI, charts, and real-time data.

---

## Features

### 📊 Dashboard (Home)
- **Real-time Stats**: Products, Campaigns, Keywords, Snapshots
- **Latest Snapshot Overview**: Period, spend, sales, ACoS
- **Action Summary**: Total actions and estimated monthly savings
- **Visual Charts**: Actions by type, Priority distribution
- **Quick Navigation**: Jump to Actions, Ownership Matrix, Categories
- **Export Downloads**: One-click download of all export files

### ⚡ Action List
- **Comprehensive Table**: All optimization actions
- **Advanced Filtering**: By channel, priority, action type
- **Summary Cards**: Total actions, savings, filters

### 👑 Ownership Matrix
- **Keyword Table**: All keywords with ownership
- **Search**: Filter keywords on the fly
- **Pagination**: Navigate thousands of keywords

### 📈 Category Performance
- **Charts**: Spend vs Sales, ACoS by category
- **Metrics**: ASINs, product groups, ROI

---

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- React Router v6
- Recharts
- Axios

---

## Quick Start

```bash
npm install
npm start
```
